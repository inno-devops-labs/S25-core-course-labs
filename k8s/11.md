### 3 . Manage Secrets with Helm:

`templates/secrets.yaml`

```yaml
apiVersion: v1
kind: Secret
metadata:
  name: token
type: Opaque
data:
  token: {{ .Values.secret.token | b64enc | quote }}
```

`templates/deployment.yaml`

```yaml
...
          env:
            - name: TOKEN
              valueFrom:
                secretKeyRef:
                  name: token
                  key: token
...
```
